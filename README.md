# 🚖 Uber Power BI Dashboard

## 📌 Project Overview
This project is an **interactive Power BI dashboard** built to analyze **Uber ride booking data** and uncover business insights related to bookings, revenue, trip distance, vehicle performance, cancellations, and customer/driver ratings.

The dashboard helps in understanding ride patterns, operational performance, and customer behavior through visually rich and easy-to-understand reports.

---

## 🎯 Objective
The main objective of this project is to:

- Track overall **Uber booking performance**
- Analyze **revenue trends**
- Measure **ride completion and cancellation rates**
- Compare **vehicle type performance**
- Identify **top pickup and drop locations**
- Monitor **customer and driver ratings**
- Support better business decision-making using data visualization

## 📊 Dashboard Features

### 1. **Booking Analysis**
- Total number of bookings
- Completed bookings
- Lost / cancelled bookings
- Booking trends over time

### 2. **Revenue Analysis**
- Total booking value / revenue
- Revenue performance by vehicle type
- Revenue trend tracking

### 3. **Trip & Distance Insights**
- Total distance travelled
- Average trip distance
- Ride distribution by route/location

### 4. **Vehicle Type Performance**
- Comparison of different vehicle categories
- Most frequently booked vehicle type
- Vehicle-wise contribution to bookings and revenue

### 5. **Location Analysis**
- Top pickup locations
- Top drop locations
- Demand concentration analysis

### 6. **Ratings Analysis**
- Average customer ratings
- Average driver ratings
- Service quality overview
